  47. def sendBulk (socket: SocketChannel, data: ByteBuffer, encode: Boolean): Unit

    Asynchrounously send the data as part of a bulk loading.

    Asynchrounously send the data as part of a bulk loading. This means that there is no guarantee when the data will be placed in the send queues. This is meant to be used in a loop where you are trying to send many byte arrays at once. Also there is no callback support here because bulk sends are lumped together into one byte buffer, so the semantics of callbacks don't really make sense
